# why do prices rise - data archive

Historical fuel price data from various sources.

## structure

- `out/` - processed data
  - `prices-history.json` - price history in standard format
  - `annotations.json` - geopolitical event annotations
- `src/` - raw source data (e.g. GasWatch PH snapshots)

## attribution

Data sourced from [GasWatch PH](https://gaswatchph.com)
